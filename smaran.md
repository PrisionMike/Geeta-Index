# Smarana

It is of 3 types:

- Bodhjanya smarana
  - The memory that you are Self. Always stays.
- Sambandhjanya smarana
  - The relations that we assume
- Kriyajanya smarana
  - The memory of duty/action. When memory becomes ingrained as actions. = अभ्यासजन्य स्मृति = अभ्यासयोग
